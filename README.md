### 3️ Python – Resume Analyzer & Job Matcher 

A smart system that analyzes resumes and matches them with job descriptions using NLP techniques.

* Skill extraction using NLP
* Match score calculation
* Missing skill identification
* Personalized learning roadmap generation
* Interactive console-based UI

####  Concepts Used

* Natural Language Processing (NLP) using NLTK
* Tokenization and stopword removal
* String processing
* Logic-based scoring system

####  Sample Workflow

* Load resume and job description
* Extract relevant skills
* Compare and calculate match percentage
* Provide suggestions and improvement roadmap

##  How to Run

###  Java Project

1. Compile the Java files
2. Run the main class
3. Follow console instructions

---

###  SQL Project

1. Import SQL file into MySQL Workbench
2. Execute queries
3. Analyze results

---

###  Python Project

1. Install dependencies:

   ```
   pip install nltk
   ```

2. Download NLTK resources:

   ```python
   import nltk
   nltk.download('punkt')
   nltk.download('stopwords')
   ```

3. Create:

   * `Resume.txt`
   * `Jobdesc.txt`

4. Run:

   ```
   python job_matcher.py
   ```
