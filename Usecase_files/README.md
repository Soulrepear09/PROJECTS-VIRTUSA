#  Multi-Use Case Projects (Java, Python, SQL)

This repository contains a collection of mini-projects use cases developed using Java, Python, and SQL. Each project demonstrates practical problem-solving skills and real-world use cases such as billing systems, fare calculation, and logistics tracking.

---

##  Project usecases Overview

### 1. CityCab Fare Calculator (Python)

A console-based application that calculates cab fares based on distance, vehicle type, and time of travel.

**Key Features:**

* Supports multiple vehicle types (Economy, Premium, SUV)
* Dynamic pricing using predefined rate structure
* Surge pricing applied during peak hours (5 PM – 8 PM)
* Trip categorization (Short, Medium, Long)
* Generates a detailed fare receipt

**Technologies Used:**

* Python
* Conditional statements and functions
* User input handling

---

### 2. Utility Bill Generator (Java)

A Java-based application that computes electricity bills using a slab-based pricing model.

**Key Features:**

* Slab-based billing system:

  * First 100 units → low rate
  * Next 200 units → medium rate
  * Remaining units → higher rate
* Tax calculation (5%)
* Input validation for meter readings
* Supports multiple users using loop
* Clean and structured bill receipt output

**Technologies Used:**

* Java
* Object-Oriented Programming (Interface, Classes)
* Scanner class for input handling

---

### 3. E-Commerce Logistics Tracker (SQL)

A database-oriented project designed to simulate order tracking and logistics management in an e-commerce system.

**Key Features:**

* Tracks order and delivery status
* Uses SQL queries for data management
* Demonstrates basic database design
* Simulates real-world logistics workflow

**Technologies Used:**

* SQL
* Database concepts (tables, queries, joins)
